Spacing Scale

With things like padding, margin, width*, height*, and position we believe that consistency is important. That is why there is a scale used in Elassus, where each step in the scale corresponds to fixed values used system wide. Those values are as follows:

Step
Value
Examples
0
0
Example: 
.p-t(0)
{ padding-top: 0; } 

.m(0)
{ margin: 0; } 

.r(0)
{ right: 0; }
1
0.25rem
Example: 
.p-t(1)
{ padding-top: 0.25rem; } 

.m(1)
{ margin: 0.25rem; } 

.r(1)
{ right: 0.25rem; }
2
0.5rem
Example: 
.p-t(2)
{ padding-top: 0.5rem; } 

.m(2)
{ margin: 0.5rem; } 

.r(2)
{ right: 0.5rem; }
3
1rem
Example: 
.p-t(3)
{ padding-top: 1rem; } 

.m(3)
{ margin: 1rem; } 

.r(3)
{ right: 1rem; }
4
2rem
Example: 
.p-t(4)
{ padding-top: 2rem; } 

.m(4)
{ margin: 2rem; } 

.r(4)
{ right: 2rem; }
5
4rem
Example: 
.p-t(5)
{ padding-top: 4rem; } 

.m(5)
{ margin: 4rem; } 

.r(5)
{ right: 4rem; }
6
8rem
Example: 
.p-t(6)
{ padding-top: 8rem; } 

.m(6)
{ margin: 8rem; } 

.r(6)
{ right: 8rem; }
7
16rem
Example: 
.p-t(7)
{ padding-top: 16rem; } 

.m(7)
{ margin: 16rem; } 

.r(7)
{ right: 16rem; }

* The width and height modules include more options than just the scale such as a 12 column grid and more.
